# 機能
## 管理者
1. 管理者ページへメールアドレスとパスワードでログイン
2. 商品の新規追加、情報編集、販売停止状態への切り替え
3. 顧客情報の閲覧、編集、退会状態への切り替え
4. 注文ステータスと商品の製作ステータスの切り替え

## 顧客
1. 顧客のユーザー登録/ログイン/ログアウト
2. 自身の配送先登録と編集
3. 自身の顧客情報の編集
4. 商品一覧の閲覧（ログイン不要)
5. 商品をカートに入れる、個数の変更、削除
6. カート内商品の注文、詳細確認
